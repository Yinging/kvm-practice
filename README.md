# kvm-practice
=================

kvm虚拟化实践，主要内容包括：
### 1、KVM虚拟化安装    
* BIOS设置
* OS安装
* KVM安装与检查

### 2、bridge方式创建windows虚拟机    
* bridge方式含义
* 宿主机网卡配置
* 虚拟磁盘创建
* 创建虚拟机
* VNC安装配置虚拟机
* 虚拟机网络配置，IP与宿主机同网段
* 使用远程连接访问虚拟机

### 2、nat方式创建windows虚拟机  
* nat方式含义
* 配置iptables端口转发，使用使用远程连接访问虚拟机

### 3、bridge方式创建linux虚拟机
* 使用ssh访问虚拟机

### 4、nat方式创建linux虚拟机
* 配置iptables端口转发，使用使用远程连接访问虚拟机

### 5、VMWare三种网络模式
* 三种模式描述，适用场景
* 实际生产部署模式
